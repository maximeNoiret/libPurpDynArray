## Building
### Compiling
Run ``make``. That's all. It'll compile a static ``.a`` library.

### Installing
To install the library (into ``/usr/local/lib`` and ``/usr/local/include``):
```bash
$ make
$ sudo make install
```

And to uninstall:
```bash
$ sudo make uninstall
```
